# game_server_plugin_management

이 저장소는 마인크래프트 서버 운영을 위한 서버 플러그인들을 관리하는 공간입니다.

Monorepo layout:
- `minecraft/plugins/card-engine`: runtime 기믹/엔진 위임 실행 플러그인
- `minecraft/plugins/card-draw`: 카드풀 추첨/적용/시즌 상태 관리 플러그인
- `minecraft/plugins/season-manager`: 시즌 카드 설계/점검 문서(info) 보관 공간
- `minecraft/resourcepacks/season_1`: season_1 서버용 리소스팩 ZIP/해시/이력 문서
