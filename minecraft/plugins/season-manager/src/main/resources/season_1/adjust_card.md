# adjust_card

- 용도: 카드 조정용 운영 문서(검토/피드백용)
- 기준 문서: `adjust_card_archive_280.md` (B120/C120/B-MOD20/C-MOD20 중심)
- 운영 카드 수: 축복 40 + 저주 40 (총 80)
- 카드 선택 단위: 카드 ID 단위(B-001~040, C-001~040)
- 카드 적용 단위: 각 카드의 T1~T4

## 공통 규격

### 효과 해석

- `받는 피해 -X%`: 최종 피해 `* (1 - X/100)`
- `받는 피해 +X%`: 최종 피해 `* (1 + X/100)`
- `%`: 기본값 대비 증감 비율
- `b`: 거리(블록)
- `s`: 시간(초)
- 동일 옵션 중복 시 가장 강한 값 1개만 적용

### 조건 충족 정의

- `전투`: 최근 6s 내 피해를 주거나 받음
- `채굴`: 최근 4s 내 블록 파괴
- `이동`: 질주/잠행 중 또는 종료 후 3s
- `환경`: 수중/용암/거미줄 접촉 또는 차원 이동 후 8s
- `조건 충족`: 위 4개 중 1개 이상 `true`

### 트리거/쿨타임

- `상시 패시브`: 항상 적용
- `조건부 발동`: 조건 충족 상태에서만 적용
- `내부 쿨타임(ICD)`: 카드별 독립 타이머
- `능동 발동`: 명시한 입력으로만 발동
- `핑 갱신`: 기본 10tick(0.5s), 별도 명시 없으면 5s 유지
